# Pendientes reales por credenciales externas

- Mercado Pago
  - `MERCADOPAGO_ACCESS_TOKEN`
  - `MERCADOPAGO_PUBLIC_KEY`
  - URL publica para webhook

- OCA
  - `OCA_API_URL`
  - `OCA_API_KEY`
  - reglas comerciales del contrato operativo

- Correo Argentino
  - `CORREO_ARGENTINO_API_URL`
  - `CORREO_ARGENTINO_API_KEY`
  - formato real de tracking y cotizacion

- Object storage
  - bucket
  - region
  - access key
  - secret key

- Auth y notificaciones
  - `NEXTAUTH_SECRET`
  - proveedor de email transaccional
